# Assets Directory

This directory contains image assets for the About page.

## Required Images

Please add the following images to this directory:

1. **Nilansh.png** - Profile image for Nilansh Gupta (Co-Founder & CEO)
   - Recommended size: 512x512px or larger
   - Format: PNG
   - Should be a professional headshot

2. **archit.avif** - Profile image for Archit Dhir (Co-Founder & CTO)
   - Recommended size: 512x512px or larger
   - Format: AVIF (or convert to AVIF)
   - Should be a professional headshot

## Image Paths

These images are referenced in the About component at:
- `/assets/Nilansh.png`
- `/assets/archit.avif`

## Notes

- Images will be displayed as circular profile pictures (rounded-full)
- Ensure images are optimized for web (compressed, appropriate file size)
- Consider using WebP or AVIF formats for better performance
